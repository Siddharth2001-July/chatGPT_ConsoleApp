# chatGPT_ConsoleApp
ChatGPT Console Application

To use this application follow the following steps : 

Pre-requisite:
1. Node.js SDK
2. npm

Steps:
1. Clone this repo
2. Run "npm i"
3. create a file name ".env" and declare "gpt_api={{Your ChatGPT api key without braces}}" inside it.
4. Run "npm run dev"
